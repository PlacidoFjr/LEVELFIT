# Prompt Comercial - LevelFit Pro

Atue como estrategista de produto, copywriter sênior e consultor comercial especialista em SaaS, healthtech, fitness, nutrição, corrida, retenção e vendas B2B.

Quero criar uma apresentação comercial para vender o **LevelFit Pro**.

## Contexto do Produto

O **LevelFit** é um app fitness gamificado premium, com foco em:

- Treino
- Alimentação
- Hidratação
- Hábitos saudáveis
- XP
- Streaks
- Conquistas
- Progresso corporal
- Acompanhamento contínuo

O usuário final usa o app diariamente para criar rotina sem pressão tóxica.

## Estrutura do Produto

### 1. LevelFit App

Área do usuário final:

- Missões diárias
- Treino do dia
- Hidratação
- Alimentação
- Progresso
- Conquistas
- Ranking
- Perfil
- Profissionais conectados
- Controle de permissões

### 2. Nutri Pro

Área para nutricionistas:

- Carteira de clientes
- Acompanhamento de adesão
- Planos alimentares
- Check-ins
- Retornos
- Alertas
- Convite por código
- Cliente autoriza permissões
- Nutricionista acompanha evolução sem depender só de WhatsApp

### 3. Run Pro

Área para coach de corrida/TAF:

- Carteira de atletas/alunos
- Treinos e planos TAF
- Agenda
- Acompanhamento de prontidão
- Carga semanal
- Convite por código
- Aluno autoriza permissões
- Coach acompanha evolução e entrega rotina pelo app

### 4. Gestão LevelFit

Área do dono:

- Controle de produtos
- Usuários
- Profissionais
- Papéis de acesso
- Revogação de acessos
- Auditoria de convites e permissões
- Separação entre Nutri Pro, Run Pro e usuário final

## Modelo de Negócio Imaginado

- Vender assinatura mensal para profissionais.
- Cada nutricionista ou coach pode ter sua própria carteira de clientes.
- O profissional usa o LevelFit como ferramenta de acompanhamento, retenção e entrega de valor.
- O usuário final pode usar o app solo ou conectado a um profissional.
- No futuro, pode haver planos por número de clientes ativos, white label, parcerias locais e marketplace de profissionais.
