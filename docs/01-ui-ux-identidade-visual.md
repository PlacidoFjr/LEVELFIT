# LevelFit - Parte UI/UX e Identidade Visual

## Papel solicitado

Atuar como designer de produto sênior, especialista em UI/UX, design systems, apps fitness, gamificação e produtos digitais premium.

## Contexto

O LevelFit é um app fitness gamificado estilo Duolingo, voltado para treino, alimentação, hidratação e hábitos saudáveis.

A prioridade máxima é a estética. A interface deve ser bonita, moderna, energética, premium, atrativa e altamente visual.

## Direção visual desejada

- Produto digital premium.
- Visual fitness tecnológico.
- Interface limpa, mas com personalidade.
- Gamificação clara e recompensadora.
- Aparência moderna, parecida com apps como Duolingo, Strava, Fitbit, Nike Training Club e apps de streak tracking.
- Evitar visual médico, frio, corporativo ou genérico.
- Evitar aparência infantil.
- Evitar aparência de landing page.
- A primeira tela deve ser o dashboard real do app.

## Paleta sugerida

- Primária: verde-limão/neon controlado.
- Secundária: azul petroleo ou grafite escuro.
- Treino: coral, laranja ou vermelho energético.
- Água: azul ciano.
- Alimentação: verde natural.
- Conquistas: amarelo/dourado.
- Fundo: grafite escuro, preto suave ou off-white premium.

- Não criar landing page.
- A primeira tela deve parecer o app real em uso.
